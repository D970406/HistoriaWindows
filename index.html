BILL GATES

Es claro que para comenzar a hablar de la historia de WINDOWS, es necesario iniciar enseñando el rostro de la persona que, de un modo u otro, le dio inicio a todo. Esta persona (Lo puede ver a la imagen de la izquierda) es Bill Gates. Willian Henry Gates III es un hombre informático, empresario y multimillonario que, al crear el sistema operativo MS-DOS en 1981, se convirtió en fundador de MICROSOFT.

En su juventud logró ingresar a escuelas prestigiosas como Lakeside y la universidad de Harvard, pero ciertas oportunidad y acontecimientos en su vida lo obligaron a abandonar este rumbo y dar comienzo a los negocios del sofware. Este negocio del software estuvo rodeado de competencia deshonesta, en el que su aliado, IBM, tomó las especificaciones de la nueva computadora APPLE II, (competidores en lo que respectaba a hardware) y lo copió y adaptó para crear PC (Personal Computer). Este PC uitilizó como sistema operativo un producto de MICROSOFT llamado MS-DOS.

MS-DOS

Los sistemas operativos de MICROSOFT nacieron en agosto de 1980 cuando compraron QDOS 0.1 a SEATTLE COMPUTER PRODUCTS. A partir de ahí y con ayuda de IBM lograron un crecimiento increible que permitió lograr que este sistema operativo evolucionara a MS-DOS y se ajustara a las necesidades del mercado respecto a sistemas operativos, ya que este MS-DOS permitia el uso de una gran variedad de programas de otros sistemas operativos también. Este hecho logró que MICROSOFT e IBM se tomaran el mercado tanto en hardware como en software en lo que respecta a los sistemas operativos y a programas.

A pesar de este gran éxito, MS-DOS tenía un gran inconveniente: solo lo usaban aquellos que tenian un gran dominio de los comandos necesarios para que funcionara. MICROSOFT no observó esta necesidad hasta que, nuevamente, APPLE tomó la delantera con un interfaz gráfico para que el manejo pudiera ser más intuitivo. En este momento MICROSOFT lo volvió a hacer en 1981: copió la interfaz gráfica de APPLE. Fue hasta noviembre de 1985 que MICROFOFT logró sacar al mercado WINDOWS 1.0

DE WINDOWS 1.0 A WINDOWS 3.51

La evolución de WINDOS 1.0 a WINDOS 3.51 Server & Workstation, que se dió entre el 20 de noviembre de 1985 y el 24 de agosto de 1995 (fecha en la que se lanzó WINDOWS 95) respectivamente, fue una muestra del crecimiento exponencial y la aceptación del mercado de MICROSOFT. WINDOWS 1.0 comenzó siendo un sistema operativo de 16 bit hasta la versión WINDOWS NT 3.1 & 3.1 ADVANCE SERVER, versión que salió en julio de 1993, que era un OS de 32 bits. Cabe aclarar que en todo este grupo de versiones de WINDOWS se uso MS-DOS como sistema operativo.

Sobre los programas con los que contaba WINDOWS 1.0 se encontraban:

  • MS-DOS Executive;
  • Calendar;
  • Notepad;
  • Windows Write;
  • Windows Paint;
  • Entre otros.
Fue en el WINDOWS 2.0 que se introdujo las primeras versiones de ofimática. que incluia:
  • Excel;
  • Word for Windows;
  • Corel Draw!;
  • Ami;
  • PageMaker;
  • Micrografx Designer.

DE WINDOWS 95 A WINDOWS 11

A partir de WINDOWS 95 se creo el modelo de interfaz gráfico que conocemos hasta el día de hoy. El sistema operativo MS-DOS, con MS-DOS 7.0 y MS-DOS 7.1 en WINDOWS 95, comenzó a ser reemplazado por sistemas operativos más completos como FreeDOS adaptado por el famoso WINDOWS XP HOME & PROFESSIONAL en octubre de 2005. También, como dato importante que señalar, WINDOWS SERVER 2003 fue el primer OS de tipo 32 & 64 bits.

Hoy en día, WINDOWS, es una de las más grandes compañias de sistemas operativos, que no solo es representativa de estos sino que también ofrece servicios de nube, hardware, consolas de videojuegos (xbox), entre otras. Los productos que actualmente se encuentran en el mercado, según la pagina web LEVENEZ, destacan los siguientes:

  • WINDOWS 10, versión 21H2;
  • WINDOWS 11, Versión 21H2;
  • WINDOWS SERVER 2022;
  • ReactOS 0.4.13;
  • XBOX ONE
